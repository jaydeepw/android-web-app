Configure Firefox

This seciton describes the Firefox settings to be configured before starting to develop extensions and addons for Firefox. Open Firefox and enter about:config in the address bar. It will warn you about changing settings, but it's ok since what you will do is only minor changes for development. You can filter the existing settings, and if any of the below settings don't exist, you can just create them.

Recommended settings

These are good to enable extension errors in the Firefox Error Console (Tools > Error Console), disable XUL caching and such.

  • javascript.options.showInConsole = true
  • nglayout.debug.disable_xul_cache = true
  • browser.dom.window.dump.enabled = true

Optional settings

These aren't necessary, but they might help you out.

  • javascript.options.strict = true
  • extensions.logging.enabled = true
